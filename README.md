## 重磅新功能

### 私密浏览模式

* 一键新建私密标签页：`Ctrl + Shift + N`
* 不保存历史、不留 Cookie，真正的无痕体验

### 固定标签页

* 右键标签页 → 「固定标签页」
* 关闭浏览器也不会消失，重要页面常驻

### 标签页智能管理

* 复制标签页
* 静音 / 取消静音单个标签页
* 一键恢复刚刚关闭的标签页：`Ctrl + Shift + T`
* 全局标签页搜索：`Ctrl + Shift + A`

### 下载与历史新体验

* 下载记录永久保存，可搜索、可直接打开
* 工具栏「↓」和「历」按钮点击即弹出最近记录
* 一键打开完整下载页 / 历史页

### 地址栏魔法命令

直接在地址栏输入以下指令即可跳转：

* `brim://settings`　`brim://history`　`brim://bookmarks`　`brim://downloads`　`brim://private`

### 更多实用工具

* 页面截图一键保存
* 打印为 PDF
* 查看页面源码
* 网页另存为 HTML
